# 🚀 Maker Hub - Ressources de Formation

Bienvenue sur le dépôt officiel des ressources pédagogiques du **Maker Hub**. Ce répertoire centralise l'ensemble des cours, documentations et tutoriels au format PDF et Markdown pour accompagner nos membres dans leur apprentissage.

## 📂 Structure du Répertoire

Le dépôt est organisé par thématiques pour faciliter la navigation :

* **⚡ Maker Hub Lab / Arduino**
    * `Niveau I` : Les bases de l'électronique et premiers pas avec l'IDE.
    * `Niveau II` : Capteurs avancés, communication et optimisation.
    * `Niveau III` : Plongée dans les architectures ARM et projets complexes.

## 📖 Comment utiliser ces ressources ?

1.  **Navigation :** Parcourez les dossiers ci-dessus pour trouver le sujet qui vous intéresse.
2.  **Lecture :** Vous pouvez lire les fichiers directement sur GitHub ou télécharger les `.pdf` pour une consultation hors-ligne.
3.  **Contribution :** Si vous repérez une erreur ou souhaitez ajouter un module, n'hésitez pas à nous contacter.
